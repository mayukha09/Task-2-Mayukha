# Project 2 – Excel Data Analysis

## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a sales dataset using Microsoft Excel. Various data analysis techniques were applied to understand trends, identify patterns, detect outliers, and summarize business performance.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts and Graphs
* Conditional Formatting

## Tasks Performed

* Data cleaning and preparation
* Summary statistics calculation
* Pivot table creation
* Identification of outliers
* Data visualization using charts
* Business insight generation

## Key Findings

* Sales performance varies across products.
* Certain products generated higher revenue than others.
* Outliers were identified and analyzed.
* Visualizations helped reveal trends and patterns within the dataset.

## Files Included

* Project-2.xlsx
* EDA Report
* Charts and Visualizations

## Conclusion

Excel provides powerful tools for data analysis and visualization. Through EDA techniques, valuable insights were extracted from the sales dataset to support business decision-making.
